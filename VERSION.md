## v1.2 주요 변경사항

### 1. 관리자 페이지 - 기한 만료 탭 추가
- 기한 만료된 사용자를 별도 탭에서 관리
- 개별/일괄 기한 연장 기능
- 전체 선택/해제 체크박스 지원

### 2. 전월 루틴 자동 복사 기능
- "전월 루틴 복사" 버튼으로 한 번에 복사
- 월별 루틴 히스토리 자동 관리
- 과거 루틴 체크 기록 보존 (Soft Delete)

### 3. AI 레이트리밋 제한 제거
- 모든 AI 기능 언제든지 사용 가능
- 사용량은 계속 추적 (통계용)
- 적용 기능: 주간/월간/연간 AI 성찰, AI 연간 목표 피드백, AI 월실천계획 제안

### 4. AI 월실천계획 가독성 개선
- 목표 사이에 빈 줄 추가
- 화살표(→) 앞에 줄바꿈 추가
- 연간목표 표시도 동일하게 개선

### 5. 버그 수정
- 날짜 선택 시 페이지 재렌더링 버그 수정
- 라우터 상태 관리 개선

### 6. 리포트 개선
- 현재 주: 월요일 ~ 오늘까지 기준 실행률 계산
- 현재 달: 1일 ~ 오늘까지 기준 실행률 계산
- 현재 연도: 1월 1일 ~ 오늘까지 기준 실행률 계산

### 7. 마이너 업데이트
- v1.2.4: 월간 실천계획 textarea 자동 높이 조절
- v1.2.3: 월말 결과 기본 템플릿 추가
- v1.2.2: 할일 삭제 확인 팝업 추가
- v1.2.1: 로그인 성능 개선, 모바일 브라우저 호환성 개선

## 버전 관리 정책

### 버전 번호 체계
- **Major (1.x.x)**: 큰 구조 변경, 호환성 깨짐
- **Minor (x.2.x)**: 새로운 기능 추가, 하위 호환성 유지
- **Patch (x.x.4)**: 버그 수정, 마이너 개선

### 릴리즈 주기
- **Major**: 필요 시 (연 1-2회)
- **Minor**: 월 1-2회
- **Patch**: 주 1-2회 (필요 시)

### 안정 버전 정책
- 각 Minor 버전은 최소 2주간 테스트 후 안정 버전으로 지정
- 안정 버전은 프로덕션 환경에서 사용 권장
- 개발 버전은 테스트 목적으로만 사용
